Rencontre sportive


Ce projet académique encadré consiste en l'élaboration d'un calendrier de rencontres sportives en utilisant deux algorithmes distincts. Le premier par un tri simple, le deuxième par l'algorithme d'Edmond. Le langage utilisé sur ce projet est Python, et s'est effectué en groupe de 4. Ce projet est le point d'entrée d'un second projet qui consistera en l'élaboration d'un site internet.

Entraînement


Ce projet personnels a pour but de compléter mes compétences avec FLutter, avec de créer des interfaces dynamiques pour une application mobile. Dans cette application, vous avez la possibilité de planifié vos sessions d'entrainements, quelque soit le type d'entrainements souhaitées et quelque soit la date. Il s'agit d'un prototype d'une application qui sera complété dans le future. Retrouvez le travail effectué sur github.
